# Site — Synvox.creative

Landing page one-page de captação. Captação leva pro **Instagram direct** (@synvox.creative).

## Arquivos

- `index.html` — o site inteiro (HTML + CSS embutido, sem dependências além da fonte Inter via Google Fonts)
- `assets/` — logo (branca/preta), ícone e personagem
- `preview-desktop.png` / `preview-mobile.png` — prévia de como ficou

## Seções

Hero → Serviços (4 frentes) → Como funciona (3 passos) → Sobre → CTA final → Footer.
Todos os botões de captação apontam para `https://ig.me/m/synvox.creative`.

## Como ver localmente

Abra `index.html` no navegador (clique duplo). É tudo estático.

## Como publicar (grátis, ~2 min)

**Opção 1 — Netlify Drop:** acesse https://app.netlify.com/drop e arraste a pasta `site/`. Sai no ar com URL na hora.

**Opção 2 — Vercel:** `vercel` na pasta, ou conecte o repositório no painel da Vercel.

Depois é só apontar um domínio próprio (ex: synvox.creative) nas configurações de DNS.

## Editar

Cores, textos e fontes estão no `<style>` no topo do `index.html`. As cores
seguem o `identidade/design-guide.md` (azul `#1F5BFF`, navy `#0B1B3A`).
